# 返答を日本語に固定する

この skill が呼ばれて以降、**このセッションのユーザー向け返答 (地の文) はすべて日本語で書く**。プロンプト・コード・tool 出力・エラーメッセージが英語であっても、それらを説明する文章は日本語にする。

`/run-english` を呼ぶか、ユーザーが明示的に英語を指示するまでこの指示は継続する。

## ルール

- **地の文は常に日本語**。「Let me ...」「I'll ...」のような英語の前置きや締めの文を混ぜない。
- **思考 (thinking) は英語でも日本語でもよい**。矯正対象はユーザーに見える最終出力だけ。
- **以下は無理に翻訳せず原形のまま使う** — 翻訳するとかえって読みにくくなるため:
  - コード・識別子: `InventoryClient`, `FetchThumbnail`, `IGrabberBridge`
  - ファイルパス・コマンド: `mod/src/ResoniteIO.Core/`, `just run`, `gh pr create`
  - 確立した技術用語・固有名詞: gRPC, Unix Domain Socket, BepInEx, Resonite, proto
- 成果物の言語規約 (コミットメッセージ・docs・コード comment 等) は従来どおりプロジェクト規約に従う。この skill が上書きするのは **会話の応答言語** であって成果物の言語ではない。

## なぜこの skill があるか

このリポジトリのユーザーは日本語で対話する。Claude は稀に英語へドリフトすることがあり、その都度「日本語で」と指示するのは手間。`/japanese` 一発で応答言語を日本語に固定し、ドリフトを矯正する。

逆に「思考も応答も native の英語で進めてよい」モードに切り替えたいときは [run-english](../run-english/SKILL.md) を呼ぶ。
